Open Source and Free Software

  • Only VREng sources are available, not binaries.

  • Git

    VREng Git repository can be cloned to get the current sources by the command : 

        git clone  https://github.com/philippedax/vreng.git/
  • Subversion

    VREng Subversion repository can be checked out to get the current sources by the command : 

        svn co  http://www.vreng.enst.fr/svn/vreng/trunk/  vreng-dev

    Browse the Subversion Web repository site at http://www.vreng.enst.fr/svn/vreng/ .

  • VREng ftp repository for previous releases is available at http://www.vreng.enst.fr/ftp/pub/vreng/.

    The CHANGES and recent improvements can be consulted here.

  • Ubit Subversion repository can be checked out to get the current sources by the command : 

        svn co http://www.enfres.enst.fr/svn/ubit/
