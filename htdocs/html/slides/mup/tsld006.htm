Utilisation

  • Programmeur d’applications, l’API:

  • Pour joindre un groupe, envoi à proxy : 3138 de : « join mcadresse mcport mcttl nomlocal portlocal »

  • Le proxy renvoie alors les paquets provenant du groupe vers client : port.

  • Pour envoyer vers le groupe : envoi a proxy : 3139 du message à envoyer au groupe.

  • Pour quitter le groupe, envoi a proxy : 3139 de :

      « drop mcadresse mcport mcttl nomlocal portlocal »
