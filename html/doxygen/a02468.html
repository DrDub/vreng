Detailed Description

This is the main class that is to animate the bones and the 3d mesh correctly...

It creates the link between the different vertices and bones and calculates the new position each time you ask

Typical use of this class : myBoneAnimator.registerMesh ( oneMesh ); myBoneAnimator.registerSkeleton ( oneSkeleton ); myBoneAnimator.compileLinkList (); ... in loop ... change skeleton current pos myBoneAnimator.animate (); myBoneAnimator.render (); ... end loop ...
