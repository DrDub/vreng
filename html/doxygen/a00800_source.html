22 /* VNC Object */
23 /* */
24 /* Thanks to that object, you can launch a VNC viewer */
25 /* into a VREng world. So, you must have a VNC server */
26 /* running on a machine (locally or remotely). */
27 /* */
28 /* The syntax to add an object into a vre file is : */
29 /* <vnc name="..." pos="..." solid="..." */
30 /* server="..." port="..." passwd="..." /> */
31 /* where */
32 /* pos <- "x,y,z,az,ax" */
33 /* server <- VNC server */
34 /* solid <- cf VReng specs (e.g. bbox,size=x,y,z) */
35 /* */
36 /* If a server was entered in the vre file, the */
