void VjcMessage::putPos ( WObject *  po)
virtual

Puts an object position.

6*32bit ((x,y,z),(az,ax,0)). This includes (x,y,z) coordinates, as well as angle data (az, ax, 0) (Y angle never used). All values are rounded to the 3rd decimal, and multiplied by 1000 so as not to have to worry about marshalling floating point values.
